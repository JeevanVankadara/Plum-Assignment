# CuraLogic — AI OPD Claim Adjudication (Frontend)

Standalone React + Vite + Tailwind CSS frontend (JSX only, no TypeScript).

## Run

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Stack
- React 18 + Vite 5
- Tailwind CSS 3
- React Router v6
